### 建议
1. 数学课看书做题就问题不大，有问题找宋浩，但是貌似近年来难度没那么低了，可能需要加大一些对于题目的思考量。
2. 计导需要把平时考试和作业题弄懂，如果需要可以自己刷刷题，力扣入门题挺友好的。
3. 道法建议提前照着网上材料画画书，方便开卷考。
4. 需要背的（特指网导）建议提前两到三周开始背，东西不是很多，背的太早可能会忘（just like me）。
